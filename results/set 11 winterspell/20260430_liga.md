Round 1 pairing was with Annika and her 🟡🔴 Amber Ruby deck using the Diary as card draw and locations combined with Raging Storm as board control. In my deck I can only go fast against this as I'd loose the long grindy game. In the first one it went exactly to this grindy game in which I had no agency anymore. In the second game I could go fast and win - while then in the last game I could not get started fast enough to gain sufficient lore advantage.
