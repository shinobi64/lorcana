# Deckwerk - Wilds Unknown PreRelease

PreRelease event for the new set 12 Wilds Unknown in the Deckwerk Bammental. 6 players showed up with high interest in the new cards. Detailed tracking via https://tcg.ravensburgerplay.com/events/577223
